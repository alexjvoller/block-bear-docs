## schedule

          <Frame>

  <img className="block" src="/images/settings-schedule.png" alt="Hero Light" />
</Frame>

### start block now

<Steps>
  <Step title="">
    if the current block hasn't started, click the start button to commence the
    block! Day 1 willl be today
  </Step>
</Steps>

### schedule block

<Steps>
  <Step title="">
    if the current block hasn't started, either manually select a start day or
    schedule the block after another. Schedule the block after another block
    will unlock the 'chain' goal option allowing you to connect goals in this
    block to the one it's scheduled too.
  </Step>
</Steps>

### complete block

<Steps>
  <Step title="">
    if the block has started and you want to immediately finish it, click the
    the complete block option.
  </Step>
</Steps>

### pause block

<Steps>
  <Step title="">
    if the block has started and you want to temporarily pause the block click
    the 'pause block' option. This will freeze the journal, and you wont be able
    add or complete any cards, or add any pins, whilst the block the is paused.
    You won't be charged for whole weekly billing increments where the block is
    paused.
  </Step>
</Steps>

### repeat block

<Steps>
  <Step title="">
    if the block has been completed and you want to repeat the block then click
    the repeat block option. This will create a copy of the block with the same
    journal and pinboard; however start points will be recalculated off the
    latest data. The new block won't automatically be started allowing you to
    schedule it when you want too.
  </Step>
</Steps>
