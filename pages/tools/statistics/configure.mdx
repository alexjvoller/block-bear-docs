## Change Chart Display Settings

Expand or collapse statistics ahat you really care about. To change display settings for a statistic:

<Steps>
  <Step title="Click the Statistic Card"></Step>
  <Step title="Click Display"></Step>
  <Step title="Select View Mode and Display Properties"></Step>
  <Step title="Choose to Set as Default or Reset to Default"></Step>
</Steps>
