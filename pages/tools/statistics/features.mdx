## Statistic Types

### Measured Trackable Statistics

All measured trackables can have the following statistics applied to give you the key insights you need.

<Tabs>
  <Tab title="Value">
    <ResponseField name="Latest">
      The latest recorded value. For example: "7 waves" for my last surf session
    </ResponseField>

    <ResponseField name="Min">
      The lowest recorded value. For example: "0 waves"
    </ResponseField>

    <ResponseField name="Max">
      The highest recorded value. For example: "8 waves"
    </ResponseField>

    <ResponseField name="Average">
      The average of your recorded values. For example: "1.2 waves"
    </ResponseField>

  </Tab>
  <Tab title="Value Over Time">
    <ResponseField name="Chart View">
      View all of your block data on a chart.
    </ResponseField>

        <ResponseField name="Trend Analysis View">
      Toggle Linear trend in the chart dispaly settings to see if you are trending up or down over the block.
    </ResponseField>

        <ResponseField name="Change Rate">
      See your average change rate and your forcast value at the end the block given your current data.
    </ResponseField>

  </Tab>
    <Tab title="Change">
     View whether the average of the values for a period is greater then the last period in both numberic and as percentage.

     For example: View weekly bodyweight average change in %.

     Supported period types: day, week, month.

    <ResponseField name="Latest">
      The latest change value. For example: "+1.5% B.W change"
    </ResponseField>

    <ResponseField name="Min">
      The lowest change value. For example: "-12.5% B.W change"
    </ResponseField>

    <ResponseField name="Max">
      The highest change value. For example: "+13.5% B.w change"
    </ResponseField>

    <ResponseField name="Average">
      The average of your chanage values. "For example: +0.25% B.w change"
    </ResponseField>

  </Tab>
  <Tab title="Change Over Time">
    <ResponseField name="Chart View">
      View all of your change data on a chart.
    </ResponseField>

        <ResponseField name="Trend Analysis View">
      Toggle Linear trend in the chart dispaly settings to see if you are trending up or down over the block.
    </ResponseField>

        <ResponseField name="Change Rate">
      See your average change rate and your forcast value at the end the block given your current data.
    </ResponseField>

  </Tab>
</Tabs>
